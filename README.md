# 📱 WA Scheduler

> **Schedule and automate WhatsApp messages with precision**

Send WhatsApp messages exactly when you want them. No manual waiting. No missed moments. Just automated messaging at your command.

---

## ✨ What It Does

**WA Scheduler** is your personal WhatsApp automation assistant. Send messages to multiple contacts on schedule or instantly, with real-time status tracking for every message.

### 🎯 Key Features

- **Multiple Recipients** — Add unlimited contacts and message them all at once
- **Scheduled Delivery** — Pick the exact time you want messages sent
- **Instant Sending** — Or send immediately when you're ready
- **Live Status Tracking** — Watch each message as it's delivered: pending → sending → sent
- **Error Handling** — Failing messages are tracked with detailed error info
- **Clean Interface** — Modern dark-theme UI designed for efficiency

## 🚀 Getting Started

### Installation

Clone the repository and install dependencies:

```bash
git clone <repository-url>
cd newyear
pip install -r requirements.txt
```

### Run It

```bash
python app.py
```

Then open [http://localhost:5000](http://localhost:5000) in your browser and start scheduling.

---

## 💻 Tech Stack

- **Backend**: Flask (Python)
- **Automation**: PyWhatKit
- **Frontend**: Vanilla JavaScript + Modern CSS
- **Architecture**: Threaded background job processing
